index.php

  • Enthält alle Input felder und buttons
  • Führt Scripts wie autocomplete.js aus

autocomplete.js

  • AJAX wird in dieser Datei verwendet
  • Vergleicht Eingabe im Wohnort Feld mit der Datenbank
  • Gibt, falls Ähnlichkeiten gefunden werden, Ergebnisse aus
  • Ruft insert_name.php per AJAX auf

insert_name.php

  • Fügt, falls nicht vorhanden, einen neuen, vom Benützer eingegebenen Ort in die Datenbank ein
  • Ausserdem werden neue Datensätze in der person Tabelle eingefügt

query_ort.php

  • Gibt autocomplete vorschläge von möglichen Orten aus.
  • Wird von autocomplete.js per AJAX aufgerufen.
  • Der Ort wird unabhängig von der Gross-/kleinschreibung verglichen
  • Falls ein passender Ort gefunden wurde, wird ein Link erstellt. Sobald man auf den clickt, füllt sich das Eingabefeld automatisch.

m133_ajax.sql

  • Enthält Tabellen: ort, person
  • Wird für diese Applikation benötigt
  • Enthält 40 Ortschaften zum vergleichen

style.css (optional)

  • Dient nur zur Verschönerung der page

killcss.js (optional)

  • Dient zum deaktivieren des CSS.
